一個單檔的互動式生日慶祝頁面，集合了歌曲獻唱、小遊戲、抽籤、祝福卡產生器等功能，並以阿珞的角色設定（黑白配色、亮黃、蝙蝠、cyberpunk、數位存在體）為視覺主軸。可在生日直播時展示，也歡迎其他粉絲打開來玩。

## ✨ 功能介紹

- **開機動畫**：終端機風格的啟動序列，呼應「我的樣貌將由你的數位足跡決定」的世界觀。
- **派對場景**：手繪的兩隻吉祥物（**阿蝠** 與 **特助**）以波浪路徑在頂部飛行，中間是插著「2」歲蠟燭的奶油蛋糕與酒席（紅酒、白酒、香檳、獺祭）。點擊兩隻吉祥物會有隱藏台詞。
- **🎤 生日快樂歌獻唱**：點擊蛋糕區的播放鈕，播放創作者親自獻唱的生日快樂歌；附**播放進度條**（可拖曳跳轉）、蠟燭火苗亮起、字幕祝福，以及阿蝠特助的引導提示。切換到任何分頁音樂都不中斷。
- **🎋 賽道法師抽籤**：共 7 款運勢籤詩，部分籤會連動阿珞的對應影片（翻唱、Live、企劃等）。抽完 7 款才會重新循環。
- **🐿️ BT欸鼠快打**：30 秒限時的打地鼠小遊戲，依分數給予傲嬌評語（注意，那是花栗鼠 🌰）。
- **💌 祝福卡產生器**：輸入名字與祝福，生成黑黃配色的專屬卡片，可截圖分享（#CetsLook）。
- **📺 觀珞影音站**：彙整直播間、YouTube 頻道、社群、贊助等入口。
- **倒數徽章**：生日前顯示倒數天數，生日後顯示已過天數。

## 🛠️ 技術說明

- **單一 HTML 檔案**，純 **HTML / CSS / JavaScript**，無任何外部框架。
- 場景與蛋糕酒席使用 **SVG** 繪製；飛行、音波、火苗等以 **CSS animation** 製作。
- 手繪吉祥物以 **Base64** 內嵌；獻唱音檔（壓縮為單聲道 m4a）亦 **Base64** 內嵌，全站無外部資源依賴，單檔即可運作。
- 音訊播放跨分頁不中斷（audio 元素置於不被切換隱藏的固定層）。
- 透過 `color-scheme: light only` 鎖定淺色模式，避免手機深色模式反轉背景配色。
- 響應式設計，桌機與手機皆可瀏覽；支援 `prefers-reduced-motion`。

### 如何使用

直接用瀏覽器打開 HTML 檔即可，無需安裝任何環境。
（直播展示時可用 OBS 的瀏覽器來源或視窗擷取。）

> 註：瀏覽器基於隱私政策禁止自動播放聲音，歌曲需手動點擊播放鈕才會響起。
