# CETRO_BIRTHDAY.exe 🎂

賽特珞 2 歲生日派對網頁｜7/3

> 一個為 VTuber 賽特珞（Cetro Timaeus）2 歲台慶打造的互動生日派對網站。

## 🌐 網址

**https://enamelu.github.io/cetro-birthday-party/**

## ✨ 功能

| 分頁 | 說明 |
|------|------|
| 🎋 賽道法師抽籤 | 點擊籤筒抽運勢，附對應賽特珞影片 |
| 🐿️ BT欸鼠快打 | 30 秒限時點擊挑戰小遊戲 |
| 💌 祝福卡產生器 | 輸入名字與祝福，生成黑黃配色專屬卡片可截圖分享至 #曬特珞 |
| 📺 觀珞影音站 | 頻道、直播、Discord 春天里民活動中心等資源快速入口 |

## 🎨 設計風格

- 開機動畫啟動進入主畫面（CETRO_BIRTHDAY.exe 主題）
- 黑黃配色、圓體字＋等寬字混搭
- 二進位代碼背景裝飾
- 生日倒數／續數日期徽章

## 🛠️ 技術

- 純 HTML / CSS / JavaScript（單一檔案，無框架）
- 字體：M PLUS Rounded 1c、JetBrains Mono（Google Fonts）
